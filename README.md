# Health Travel App（仮）

## 1. サービス概要
このサービスは、日々の運動量をカロリーから距離に変換し、日本を旅するように健康管理ができるアプリです。  
運動すればするほどキャラが成長し、ご当地アイテムを集めながら旅が進むため、楽しみながら運動習慣を継続できます。  
運動不足に悩む人が、ゲーム感覚で運動を続けられる仕組みを提供します。

---

## 2. このアイデアはどこから生まれたか

### 2-1. きっかけとなった体験・感情
仕事や日常が忙しく、運動する機会がなかなか取れず、ジムに通っても長続きしない時期が続いていた。  
「運動しなきゃ」と思っていても、気づけば運動不足になっている自分に焦りやモヤモヤを感じていた。  
ただ目標を立てるだけでは続かず、楽しさや達成感がないと継続できないという実感があった。

### 2-2. なぜそれが「気になった」のか
自分はゲーム性や成長要素があるとモチベーションが上がり、楽しみながら継続できるタイプだと感じていた。  
「運動にもゲームのような仕組みがあれば続けられるのでは？」という思いが強くなり、  
運動不足を解消したい気持ちと、楽しさがないと続かない性格が結びつき、このアイデアが生まれた。

---

## 3. 課題の整理

### 3-1. 表に見えている困りごと
- 運動を続けるモチベーションが保てない  
- 記録アプリは味気なくて続かない  
- 運動不足を自覚しているが行動に移せない  

### 3-2. 本当に解決したい課題
- 運動が習慣化しない  
- 達成感や楽しさが感じられない  
- 運動の成果が実感しづらい  

→ **本当に解決したい課題：  
「運動の成果を実感し、楽しみながら継続できる仕組みがない」**

---

## 4. 想定ユーザーについて

### 4-1. 想定ユーザー（具体化）
**40代・デスクワーク中心の会社員（過去の自分）**

- 1日の大半をPCの前で過ごし、運動不足を自覚している  
- 仕事の疲れで運動が続かず、休日は家で過ごしがち  
- 健康診断で「運動不足」「体重増加」を指摘され焦りを感じている  
- 記録アプリは味気なくて続かない  
- ゲームやキャラ育成が好きで、成長が見えると続けやすい  
- 「運動＝義務」ではなく「運動＝楽しみ」に変えたい  
- 過去の自分と同じように、運動習慣をつけたいが続かない人を助けたいという思いがある  

### 4-2. 自分とユーザーの距離
→ 過去の自分そのもの。  
運動不足に悩み、継続できなかった経験があるため、課題を深く理解している。

### 4-3. 実際に使われる場面（具体化）
- **朝**：通勤前に昨日の運動を記録し、旅が進んだことを確認  
- **昼休み**：10分歩いたので記録し、「あと80kmで次のエリア」と進捗を見てモチベUP  
- **夜**：疲れて運動できなかった日でも、旅マップを見て「明日は少し歩こう」と前向きに  
- **休日**：30分の運動でキャラが成長し、ご当地アイテムが手に入る  

### 4-4. 現在の代替手段と乗り換え理由
**現在使っているもの**
- iPhoneヘルスケア（歩数のみ）
- Nike Run Club（ラン記録）
- Google Fit（健康データ記録）

**課題**
- 数値が並ぶだけで楽しくない  
- 運動が続かない  
- 達成感が弱い  

**なぜ乗り換えるのか**
- 運動すると旅が進む“物語性”がある  
- キャラが成長し、努力が可視化される  
- ご当地アイテム収集で継続したくなる  
- 数値管理ではなく「楽しさ」で運動を続けられる

## 5. 既存サービス・競合調査（体験ベース）

### 5-1. 競合比較表

| サービス名 | できること | できないこと | 本アプリとの違い（体験の差） |
|------------|------------|--------------|------------------------------|
| ドラクエウォーク | 歩数で冒険、RPG、キャラ育成、ご当地要素 | 運動量の記録や健康データとの連動 | 歩くことが中心で「運動習慣の形成」が目的ではない |
| ポケモンGO | 位置情報ゲーム、収集、探索 | 運動量の可視化、継続的な健康管理 | 運動はゲームの副産物であり、健康目的ではない |
| Habitica | 習慣化×RPG、タスク達成で成長 | 運動に特化した体験、旅の概念 | 生活全体の習慣化が目的で、運動に特化していない |
| Nike Run Club | ラン記録、コーチング、分析 | ゲーム性、旅、キャラ成長 | 健康管理が中心で、楽しさや継続の仕掛けが弱い |

### 5-2. 差別化を一文で
**「運動量がそのまま“旅の進行”と“キャラ成長”に変わることで、運動がゲームの物語として続いていく体験」を提供する点が唯一無二です。**

## 7. MVPで実装する機能（明確化）

### 1. 運動記録機能
- 種目、時間を入力して運動を記録できる  
- カロリー計算を自動で実行  

### 2. カロリー → 距離変換
- 記録したカロリーを距離(km)に変換  
- 変換式：距離 = カロリー / 50  

### 3. 旅の進捗表示
- 累計距離に応じて現在地を表示  
- 100kmごとに次のエリアへ進行  

### 4. キャラクター成長
- 運動記録1回につき10exp  
- Lv1〜Lv5まで成長  

### 5. マイページ
- 今日の運動量  
- 累計距離  
- 現在のエリア  
- キャラのレベル  

### 6. ログイン / 新規登録
- 自分のデータを継続管理できる  

---

## 8. 運動記録の仕様（カロリー計算）

### カロリー計算式（MVP）
消費カロリー = METs × 60kg × (運動時間 / 60) × 1.05

### 種目ごとの METs（例）
| 種目 | METs |
|------|------|
| ウォーキング | 3.5 |
| ランニング | 7.0 |
| サイクリング | 6.0 |
| 筋トレ | 5.0 |
| ヨガ | 2.5 |

### カロリー → 距離変換
距離(km) = カロリー / 50

---

## 9. 旅の進行・キャラ成長の数値仕様（MVP）

### 旅の進行
- 1 kcal = 0.02 km  
- 100 km ごとに次の地域へ進行  
- MVPでは最初の3エリアのみ実装  

### キャラクター成長
- レベル：Lv1〜Lv5  
- 経験値：運動記録1回につき10 exp  
- レベルアップ条件：  
  - Lv1 → Lv2：50 exp  
  - Lv2 → Lv3：100 exp  
  - Lv3 → Lv4：150 exp  
  - Lv4 → Lv5：200 exp
